about=This plugin integrates the IBM-NASA Prithvi EO 2.0 Geospatial Foundation Model (GFM) to automate environmental hazard detection, specifically focusing on flood mapping and burn scar identification.

        The tool utilizes a hybrid Client-Server architecture:
        - <b>Lightweight Frontend:</b> Seamlessly maps multi-spectral bands (Blue, Green, Red, NIR, SWIR) within the QGIS environment.
        - <b>High-Performance Backend:</b> Offloads heavy AI inference to a dedicated local or remote Python process.

        <b>Note:</b> This plugin requires a running backend server. For installation and setup instructions, please visit <a href="https://github.com/leonardijulia/hazard_detection">the GitHub Repo</a>

        Szwarcman, D., Roy, S., Fraccaro, P., et al. (2024). Prithvi-EO-2.0: A Versatile Multi-Temporal Foundation Model for Earth Observation Applications. arXiv preprint arXiv:2412.02732. Available at: <a href="https://arxiv.org/abs/2412.02732">https://arxiv.org/abs/2412.02732</a>
